🏡 Auberge — Site Web (Work in Progress)
📌 Présentation

Ce projet est le développement du site web d'une auberge.

L'objectif actuel est de présenter une première version du design de la page d'accueil.
Il s'agit d'une maquette fonctionnelle encore en cours d'évolution.

🚧 État actuel du projet

Le projet est en cours de développement.

✅ Actuellement

Page d'accueil construite

Structure globale en place

CSS de base implémenté

Routing configuré


🔜 À venir

Ajustement des tailles typographiques

Traductions complete en :

🎨 À propos du design

Certaines décisions visuelles évolueront :

La police n'a pas encore été choisie

Les tailles de texte sont provisoires

Les photos et videos utilisées ne sont pas définitives

Le design final dépendra fortement du choix typographique et des contenus définitifs.
